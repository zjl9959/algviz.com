本地安装

在开始安装之前，请先检查您的 Python 版本号是否大于等于 3.7。 如果小于的话，请更新您的 Python 版本！ 之后请更新您的 pip 版本至最新版：

python -m pip install --upgrade pip

第一步：安装 Jupyter Notebook

您可以选择以下任意一种方式进行安装：

方式一：Jupyter

参考 Jupyter 官方提供的 安装教程：

pip install jupyterlab

或是

pip install notebook

方式二：VSCode

使用 VSCode 提供的 Jupyter 拓展插件。

方式三：Anaconda

安装 Anaconda，使用其自带的 Jupyter Notebook。

第二步：安装 Graphviz

您可以直接从 Graphviz 的官方网站免费下载安装。

安装完成后，请将 Graphviz 的可执行文件路径添加到系统的环境变量中！

第三步：安装 Algviz

pip install algviz

安装验证

要验证 algviz 是否安装成功，您可以从 Github 下载单元测试代码，然后执行命令：

python tests/run.py

如果您看到了下面的执行结果：

Congratulations, everything is OK !!!

那么您的本地配置环境一切正常，否则，您可以从这里报告您的错误。

安装完成之后，您可以参考使用教程来学习如何使用 Algviz 让自己的算法动起来！
