只需简单几步 让算法动起来

Algviz 是一个用于生成算法可视化动画的 Python 库，您只需在代码中引入少量的接口，即可看到算法的运行动画！

  • 冒泡排序算法可视化

  • 合并链表过程可视化

  • 字典树构造过程可视化

  • 拓扑排序算法可视化


    免费安装


    支持多种数据结构

    Algviz 中定义了多种数据结构对象，以支持丰富的动画效果，您可以像使用 Python 内置对象一样来使用它们。

    向量 表格 树 拓扑图

    将动画嵌入笔记中

    您可以在 Jupyter Notebook 中使用 Algviz 生成动画，并在笔记本中实时预览算法的运行过程。
